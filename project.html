Aim

What this lab achieves

Objective

To understand and apply the fundamental principle of partial differentiation — treating all other variables as constants while differentiating with respect to one variable.


Learning Outcomes

  • Understand the concept of "treating a variable as constant" in partial differentiation.
  • Compute partial derivatives ∂f/∂x and ∂f/∂y for multivariable functions.
  • Use the interactive simulator to verify manual calculations.
  • Evaluate understanding via pretest and posttest assessments.
  • Apply partial differentiation to standard engineering problems.

Apparatus

This virtual lab requires only a web browser. No physical setup is needed.

Theory

Variable as a Constant — Key Principles

What is Partial Differentiation?

For a function of multiple variables f(x, y), the partial derivative with respect to x is found by treating y as a fixed constant and differentiating normally with respect to x.

∂f/∂x → differentiate w.r.t. x, keep y constant
∂f/∂y → differentiate w.r.t. y, keep x constant

Worked Example

Let f(x, y) = x²y + 2xy²

∂f/∂x = 2xy + 2y²   (y treated as constant)
∂f/∂y = x² + 4xy   (x treated as constant)

Rules Applied

  • Power rule: ∂(xⁿ)/∂x = nxⁿ⁻¹
  • Constant rule: ∂(c)/∂x = 0
  • Product rule: when both x and y appear in a term, use standard product rule keeping the other variable constant.
  • Chain rule: ∂/∂x [f(g(x,y))] = f'(g) · ∂g/∂x

Notation

Partial derivatives are written as ∂f/∂x or fₓ. The symbol ∂ (curly d / del) distinguishes partial from ordinary derivatives.
